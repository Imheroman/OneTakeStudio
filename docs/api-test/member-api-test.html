테스트 흐름

  • 초대 발송: HOST/MANAGER가 스튜디오에 멤버 초대
  • 초대 확인: 초대 대상자가 받은 초대 목록 조회
  • 초대 수락/거절: 초대 대상자가 초대를 수락하거나 거절
  • 스튜디오 초대 관리: HOST/MANAGER가 보낸 초대 목록 조회 및 취소
  • 멤버 관리: 멤버 목록 조회, 역할 변경, 강퇴
  • 스튜디오 탈퇴: MANAGER/GUEST가 스튜디오 탈퇴

받은 초대 관리

GET 내가 받은 초대 목록 /api/invites/received

현재 로그인한 사용자가 받은 대기중인 초대 목록을 조회합니다.

POST 초대 수락 /api/invites/{inviteId}/accept

POST 초대 거절 /api/invites/{inviteId}/reject

스튜디오 초대 관리 (HOST/MANAGER)

POST 멤버 초대 /api/studios/{studioId}/members/invite

GET 스튜디오 초대 목록 /api/studios/{studioId}/invites

DELETE 초대 취소 /api/studios/{studioId}/invites/{inviteId}

멤버 관리

GET 멤버 목록 조회 /api/studios/{studioId}/members

PATCH 멤버 역할 변경 (HOST only) /api/studios/{studioId}/members/{memberId}

POST 멤버 강퇴 (HOST/MANAGER) /api/studios/{studioId}/members/{memberId}/kick

DELETE 스튜디오 탈퇴 /api/studios/{studioId}/members/me

HOST는 탈퇴할 수 없습니다. MANAGER/GUEST만 탈퇴 가능합니다.
