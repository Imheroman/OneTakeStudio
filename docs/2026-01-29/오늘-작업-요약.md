## 주요 변경 사항

### 인프라 구성 (최신 상태)
```
┌────────────────────────────────────────┐
│           docker-compose               │
├────────────────────────────────────────┤
│  MySQL 8.0       ─ 3306                │
│   ├─ core_db     (Core Service)        │
│   └─ media_db    (Media Service)       │
│                                        │
│  Redis 7         ─ 6379                │
│   └─ Streams (메시지 브로커)            │
│                                        │
│  LiveKit         ─ 7880, 7881          │
│   └─ WebRTC SFU                        │
└────────────────────────────────────────┘
```
