# 미디어 설정 기능 구현

> 작성일: 2026-01-29
> 기능명세서: S04 (장치 자동 초기화 및 감지), S10 (실시간 하드웨어 제어)

## 개요

Media Service에 사용자 미디어 설정(영구 저장)과 세션 미디어 상태(실시간 관리) 기능을 추가했습니다.

---

## 구현된 기능

### 1. 사용자 미디어 설정 (UserMediaSettings)
- 기본 카메라/마이크/스피커 디바이스 저장
- 비디오/오디오 품질 설정 (LOW, MEDIUM, HIGH, ULTRA / STUDIO)
- 노이즈 캔슬링, 에코 캔슬링 설정
- 기본 볼륨 레벨 (0-100)

### 2. 세션 미디어 상태 (SessionMediaState)
- 실시간 비디오/오디오 On/Off 토글
- 뮤트 상태 토글
- 현재 사용 중인 디바이스 추적
- 볼륨 레벨 조절

### 3. Stream 연동
- 스트림 참가 시 미디어 상태 자동 초기화
- 스트림 퇴장 시 미디어 상태 자동 종료

## Stream 연동 흐름

```
[프론트엔드]
    │
    ▼
POST /api/v1/media/stream/join
    │
    ├─→ StreamSession 생성 (CONNECTING)
    ├─→ LiveKit 토큰 발급
    └─→ SessionMediaState 자동 초기화 (사용자 기본 설정 적용)

    │
    ▼
POST /api/v1/media/stream/{studioId}/leave
    │
    ├─→ StreamSession 종료 (DISCONNECTED)
    ├─→ LiveKit 참가자 제거
    └─→ SessionMediaState 자동 종료
```

# 통합 채팅 시스템 구현 (S11)

## 개요

YouTube, Twitch, 치지직(CHZZK) 등 외부 플랫폼 채팅을 수집하여 통합 관리하는 시스템입니다.
OneTakeStudio는 OBS처럼 방송을 송출하는 도구이므로, 시청자들은 각 플랫폼에서 채팅을 입력합니다.
이 채팅을 수집하여 스튜디오 내에서 통합 표시합니다.

---

## 구현된 기능

### 1. 내부 채팅 시스템
- 메시지 저장 및 조회 (DB 영구 저장)
- WebSocket STOMP 기반 실시간 브로드캐스트
- 플랫폼별 메시지 구분 (INTERNAL, YOUTUBE, TWITCH, CHZZK)
- 슈퍼챗/도네이션 지원

### 2. 외부 플랫폼 연동 구조
- `ExternalChatClient` 인터페이스 기반 확장 가능한 설계
- 플랫폼별 클라이언트: YouTubeChatClient, TwitchChatClient, ChzzkChatClient
- 5초 주기 폴링으로 메시지 수집 (스케줄러)
- 연동 시작/종료 API

## 외부 플랫폼 연동 구조

### 연동 흐름
```
[방송 시작]
    │
    ▼
POST /integration/{studioId}/youtube/start
    │
    ├─→ YouTubeChatClient.connect(credentials)
    └─→ activeIntegrations에 등록

    │
    ▼
[5초 간격 스케줄러 실행]
    │
    ├─→ YouTubeChatClient.fetchNewMessages()
    ├─→ ChatService.receiveExternalMessage()
    └─→ WebSocket 브로드캐스트 (/topic/chat/{studioId})

    │
    ▼
[방송 종료]
    │
POST /integration/{studioId}/stop-all
    │
    └─→ 모든 클라이언트 disconnect()
```

### 플랫폼별 연동 방식

| 플랫폼 | 연동 방식 | 인증 |
|--------|----------|------|
| YouTube | REST API Polling | OAuth 2.0 (accessToken, refreshToken) |
| Twitch | IRC over WebSocket | OAuth Token + Channel Name |
| 치지직 | WebSocket | Channel ID (비로그인 가능) |
