인증 (Auth API)

POST/api/auth/login

로그인 후 JWT 토큰 발급

회원가입 (3단계)

POST/api/auth/send-verification - 1단계: 인증 코드 발송

POST/api/auth/verify-email - 2단계: 이메일 인증

POST/api/auth/register - 3단계: 회원가입 완료

Destination API (송출 채널 관리)

GET/api/destinations

내 연동 채널 목록 조회

POST/api/destinations

신규 송출 채널 등록

GET/api/destinations/{destinationId}

단일 채널 조회

PUT/api/destinations/{destinationId}

채널 정보 수정

DELETE/api/destinations/{destinationId}

채널 연동 해제

Internal API (서비스 간 통신용)

POST/api/destinations/internal/batch
