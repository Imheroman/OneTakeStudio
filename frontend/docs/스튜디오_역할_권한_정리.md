# 스튜디오 역할(role) 및 권한 정리

## 역할별 정의

| 역할        | 설명                         | 권한                                                           |
| ----------- | ---------------------------- | -------------------------------------------------------------- |
| **HOST**    | 스튜디오 소유자(주인)        | 장면 전환, 장면 생성/이름변경/삭제, 편집, 소스 추가, 추천 수신 |
| **MANAGER** | 장면 생성·추천 담당 (도우미) | 장면 생성, 장면 이름 변경, 편집, 소스 추가, 장면 추천          |

## GUEST 제거

송출 서비스에서는 시청이 **송출 플랫폼**(YouTube, Twitch 등)에서 이루어지므로,  
스튜디오 내 "시청 전용" GUEST 역할은 불필요하여 제거했습니다.

초대 시 모든 멤버는 **MANAGER** 역할로 초대됩니다.

## 권한 매트릭스

| 기능                | HOST | MANAGER |
| ------------------- | ---- | ------- |
| 장면 전환           | ✅   | ❌      |
| 장면 생성           | ✅   | ✅      |
| 장면 이름 변경      | ✅   | ✅      |
| 장면 삭제           | ✅   | ✅      |
| 소스 추가           | ✅   | ✅      |
| 장면 추천 (방송 중) | -    | ✅      |
| 추천 수신·확인      | ✅   | ❌      |
